INTRODUCTION & CONTEXTE
-----------------------
Les programmes deviennent réellement utiles lorsqu’ils peuvent prendre
des décisions et répéter des actions. Le contrôle de flux permet à un
programme de :

- Exécuter différents blocs de code selon des conditions.
- Répéter des instructions grâce aux boucles.
- Combiner des conditions logiques pour modéliser des situations réelles.

Ce projet se concentre exclusivement sur le contrôle de flux en Python,
en utilisant les éléments suivants :

- if, elif, else
- Opérateurs de comparaison
- Logique booléenne
- Boucles while
- Boucles for avec range()

Les activités proviennent du projet "Python - if/else, loops, functions",
mais seules celles liées au contrôle de flux ont été conservées.

OBJECTIFS PEDAGOGIQUES

A la fin de ce projet, vous serez capable de :
